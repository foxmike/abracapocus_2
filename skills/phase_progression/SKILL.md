# Phase Progression Skill

- **Purpose**: Determine when a project phase can advance, carry unfinished tasks forward, and align with multi-phase plans.
- **Inputs**: active phase state, completion criteria, task histories.
- **Outputs**: decisions on phase transitions and recommended next tasks.
- **Usage**: planning and management agents rely on this skill when creating or updating plans.
- **Future Work**: integrate with verification gating (e.g., require all checks to pass before advancing).
