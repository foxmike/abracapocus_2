# Documentation Generation Skill

- **Purpose**: Produce operator-readable docs: plan updates, execution reports, change summaries.
- **Inputs**: plan/task metadata, backend outputs, review/verification results.
- **Outputs**: Markdown or JSON documents saved under `docs/` or `reports/`.
- **Usage**: research and reviewer agents use this skill to keep human operators informed.
- **Future Work**: auto-publish docs to knowledge bases or PR templates.
